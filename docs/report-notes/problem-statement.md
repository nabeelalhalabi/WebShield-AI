# Problem Statement

Users regularly encounter offensive text, explicit imagery, violent visuals, and manipulative promotional banners while browsing the web. Standard browser tools rarely let users define fine-grained detection categories and actions for these risks.

WebShield addresses this problem by combining pretrained AI moderation models with user-controlled filtering policies inside a browser-extension workflow. The system focuses on practical, high-value detection cases rather than attempting perfect fraud detection or universal moderation.
